# ComputerVisionProjects
Παρακάτω, επισυνάπτονται αρχεία Python όπου χρησιμοποιούνται διάφορες τεχνικές της Μηχανικής Μάθησης (SVM,KNN) καθώς και συνελικτικά δίκτυα για την ταξινόμηση εικόνων(image classification), τη σημασιολογική κατάτμηση (semantic segmentation) και την εξαγωγή κατάλληλων περιγραφικών (feature extraction)
